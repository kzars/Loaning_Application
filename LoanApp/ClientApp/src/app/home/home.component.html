ASPCore Application With Angular

Welcome to single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

This application contains 3 main parts:

  • Person list. Displaying all Person names, surnames from DB.
  • Loans list. Displaying all Loans with borrower and loaner ID's and amount loaned.
  • Pay Back's list. Displaying all Pay back transactions with loan ID and amount payed back.
